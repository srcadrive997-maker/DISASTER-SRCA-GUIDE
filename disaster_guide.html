🏷️ MCI Triage Quick Reference (START / JumpSTART)
🔴 RED
IMMEDIATE
Life-threatening, high survival potential. Treat first.
🟡 YELLOW
DELAYED
Serious injuries. Can wait without increasing mortality.
🟢 GREEN
MINIMAL
Minor injuries. Walk-and-wait. Self-care possible.
⚫ BLACK
EXPECTANT
Unlikely to survive with available resources.
⚡ START Triage — RPM Assessment
R — Respirations: <10 or >29/min → RED | 10-29/min → continue
P — Perfusion: Radial pulse absent OR cap refill >2 sec → RED
M — Mental Status: Cannot follow simple commands → RED
👶 JumpSTART (Pediatric <14 yrs)
Breathing: Apneic → open airway → if still apneic + no pulse → BLACK | if pulse present → 5 rescue breaths → if breathing resumes → RED
RR: 15-45/min → continue | <15 or >45 → RED
Perfusion: Peripheral pulse present → continue | absent → RED
Mental Status (AVPU): Alert/Verbal/Pain (appropriate) → YELLOW | Pain (inappropriate)/Unresponsive → RED
